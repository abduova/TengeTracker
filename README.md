# TengeTracker

TengeTracker is a personal budget and expense tracker web app designed to help users track their daily expenses, manage different financial accounts, and set monthly spending goals.

##  Team Members

- Akbota Abduova
- Dinmukhamed Baitaliyev
- Alima Toleugali

##  Main Features

- Record, view, edit and delete income and expenses    
- Monitor financial habits  
- Manage multiple money sources (cash, bank accounts, digital wallets).
